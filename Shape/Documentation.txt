Currently, the Shape v2.0 Assembler is very limited in scope.

-----------------------------

Sections:

.code
.data

-----------------------------

Labels:

 - A label ends with :

entry:
string1:
string2:
loop:

-----------------------------

Instructions:

ret
 - Return call within the program (not much use yet)

yield
 - Calls the host API yield function

nop
 - Do nothing

loadbase
 - Loads the current program image base address into register b
 - Necessary to call for label-based memory access (ex: [label]) as global data is addressed in relation to the global image
 - Should call loadbase near the start of the program before using lea or mov (with label)

exit
 - Calls the host API exit function (terminates program)

speak <label>  |  say <label>      | >> (interchangeable)
 - Prints a null-terminated string stored in .data at the given label

show
 - Prints the value in register a

hear
 - Await user input (single char right now)

-----------------------------

Memory:

(<imm> is an immediate)

mov a, <imm>
 - Set a

mov a, [label]
 - Load the contents stored at the data label

mov [label], a
 - Store the current value in register a into a data label

lea a, <label>
 - Load the runtime memory address of a data label into register a
 - Loads address itself, not the value stored in the label

-----------------------------

Arithmetic:

(<imm> is an immediate)

add a, <imm>
 - a + imm

sub a, <imm>
 - a - imm

mul a, <imm>
 - a * imm

div a, <imm>
 - a / imm

cmp a, <imm>
 - Compare a to imm


Immediates can be integers or character literals
ex:
10
-4
'1'
'\n'

-----------------------------

Control Flow:

jmp <label>
 - Unconditional jump to label

jz <label>
 - Jump to label if zero

jnz <label>
 - Jump if not zero

je <label>
 - Jump if equal

jne <label>
 - Jump if not equal

-----------------------------

Data:

asciz "string"
 - Stores a null-terminated string
 - Supports  \n  \t  \\  and  \"

dd
 - Stores a 32-bit value
   - dd 41
   - dd -4
   - dd 'C'
 - Can be used for global integers, constants, or regular variables


Comments:

 - Comments start with ;
; a comment looks like this


Requirements:

 - Code must contain an entry label (entry:) within .code
 - speak must refer to a .data label


-----------------------------


Building and Summoning a Golem:

shape hello.asm hello
summon hello.glm

 - The shape command builds a golem executable from the shape assembly code
 - The summon command runs the executable golem



Examples of valid golems are shown in the examples folder
